# MIDI Message Gate Utility

## What is This?

This small utility turns your M5Stack into a MIDI message gate.
- Button A enables/disables MIDI message flow.
- Button C sends out all-notes-off and all-sound-off to all the MIDI channels

Here's [some demo video clip](https://twitter.com/ichiohta/status/1386532099503525892).

## Requirements

- [Arduino MIDI Library](https://playground.arduino.cc/Main/MIDILibrary/)
- Some compatible MIDI module - This module was developed and tested with necobit MIDI module for M5Stack
